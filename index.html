ETIQUETAS SEMANTICAS

Las etiquetas semánticas, también conocidas como "etiquetas HTML semánticas", son una forma de marcar el contenido de una página web con información sobre su significado o propósito. Estas etiquetas proporcionan información adicional a los motores de búsqueda y a otros programas que analizan y procesan el contenido web. Algunos ejemplos de etiquetas semánticas en HTML incluyen "header", "nav", "article", "section", "aside", "footer", "main", "figure", "figcaption", entre otras. } Cada una de estas etiquetas se utiliza para identificar una sección específica del contenido de la página y ayudar a los motores de búsqueda a entender cómo está estructurado el contenido.

ETIQUETAS META

Las etiquetas meta, también conocidas como "meta tags", son elementos del código HTML que proporcionan información sobre una página web a los motores de búsqueda y a otros programas que la procesan. Estas etiquetas no son visibles para los usuarios que visitan el sitio web, pero proporcionan información importante sobre el contenido de la página y su relevancia para los motores de búsqueda. Las etiquetas meta pueden incluir información sobre la descripción del contenido, las palabras clave relevantes, el autor de la página, la fecha de creación, el idioma utilizado, el tipo de contenido y otros metadatos que ayudan a los motores de búsqueda a comprender el contenido y mostrar resultados relevantes a los usuarios. Algunos ejemplos de etiquetas meta incluyen: La etiqueta "description", que proporciona una breve descripción del contenido de la página. La etiqueta "keywords", que incluye una lista de palabras clave relevantes para el contenido de la página. La etiqueta "author", que identifica al autor de la página. La etiqueta "viewport", que controla la forma en que se muestra el contenido en diferentes dispositivos.

PARA QUE SON LAS ETIQUETAS SEMANTICAS

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.
